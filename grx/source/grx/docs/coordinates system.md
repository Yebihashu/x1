# Used coordinates system:
GRX coordinates system is a left hand coordinates system:

### Axes directions:
x - right
y - up
z - front

### Rotation directions:
(left hand rule)
roll  - around z - positive is counter clock wise
pitch - around x - positive is down
yaw   - around y - positive right 

### Order of rotations:
1. roll
2. pitch
3. yaw
